Page 3: 3 - Boids

This is a boid project. The boids bounce off of the boundaries of the box, obstacles, and each other, and the user can determine the number of boids, the speed of the boids, and how much their direction changes upon impact with the wall.

I am proud of the boid design. It's simple yet elegant and effective. I am also happy with how the controls for the number of boids and the speed turned out, because these controls, in my opinion, are the most fun.

This applies the following:

  • Many objects moving at once:   the boids move individually
  • Non-trivial 2D Canvas animation:   the boids bounce off the edges
  • Use of basic transformations:   translated and rotated the boids
  • An SVG that uses instancing, arcs, and curves:   there is an arc in the boids' tails, a curve in the boids' wings, and <use> tags are used for the boids' wings.
